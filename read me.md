# Project Title
Webcomic Website
## Project Description
A website for a "webcomic" I made

## Intended User or Player
For users to read a page from a webcomic I'm planning to make, flip to the next page, and have the option to go to other parts of the webcomic

## User / Player Goal
Readers of 14 years of age and above who are looking to laugh for a bit.

## Inspiration Interfaces
Lackadaisy  Website
https://lackadaisy.com/comic.php?comicid=161
I noticed it has a really nice dropdown menu that lets you continue to the next part of the webcomic; it really helps if you are on a page but want to go to another one and are sort of near it in the story's continuity. I also like the color scheme of the website and how it fits with the comic. I'm going to implement both of these elements if I'm skilled enough to do so.
Explosm's Website
https://explosm.net/
I noticed the left and right arrows that lead you to the next comic. I also like that these arrows are located at the top and the bottom of the comic. This facilitates navigation. I also like the random dice, which I think is a really fun feature to play with if you want a random funny comic.

## Design Choices
- Layout: The page is centered around the comic so that's the main focus, there is a sidebar to navigate through the seasons and a top menu to nagivigate through the page for additional content.
- Color: A simple black and white color scheme, so people can know where to click, as well as a bit of yellow with the seasons title, to draw peoples attention and to match with the banner color.
- Typography: Arial was chosen because it looks simple and profesional.
- Visual hierarchy: The big banner is the first thing that's seen, so that everyone knows what comic they are reading, after that you see the menu, then you see the big comic which is the focal point, then you might see the arrows if you want to see the next comic or the small lettered seasons tab to check more comics.
- Feedback states:Navigate links change color when you hover over them, and the dice and the arrows change size when hovering over the,
- Responsive design: Whenever it is scaled to bigger or smaller the size of the comic and most options remain the same.
